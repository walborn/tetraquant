# Методы в компонентах

https://jaketrent.com/post/naming-event-handlers-react/

В компонент все действия передаются через `on*`
При этом, если действие объявлено внутри компонента, то обозначаем его через `handle*`

```tsx
onChangeFormField
```
1. Начинать нужно с handle
2. Можно называть просто handleChange, так как внутри формы это имя будет уникальным и добавлять FormField будет избыточным
